# Data

This directory contains the data collected and curated by the Museum Closure project team and used for the data analyses undetaken.

- `closure_data`: data in CSV format dumped from our Neo4j database of collection dispersal data.
- `report_data_01_08_2025`: summary tables based on `closure_data` in XLSX format used in the preparation of the report Candlin _et al_ _Collections From Closed Museums in the UK 2000-25: A report on the data_.
- `dispersal_sheets`: snapshots in XLSX format of data concerning the flow of objects away from closed museums. The latest dispersal sheet and Mapping Museums snapshot can be used with the `sheet_to_graph` tool to populate a Neo4j database of collection dispersal data.
- `mapping_museums_data`: snapshots in CSV format of the Mapping Museums database.
